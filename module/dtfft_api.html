private function dtfft_get_local_sizes_c(plan_ptr, in_starts, in_counts, out_starts, out_counts, alloc_size) result(error_code) bind(C)

Returns local sizes, counts in real and Fourier spaces and number of elements to be allocated for in and out buffers, C/C++ interface. It is necessary for R2C transform to allocate out buffer with alloc_size number of elements. Total number of bytes to be allocated : - C2C: 2 * alloc_size * sizeof(double/float) - R2R: alloc_size * sizeof(double/float) - R2C: alloc_size * sizeof(double/float)
