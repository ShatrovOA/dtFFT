dtfft_reshape_handle_datatype dtfft_reshape_handle_datatype.F90

This module describes reshape_handle_datatype class It executes either transposition or reshaping from bricks to pencils and vice versa using MPI_Ialltoall(w) with custom MPI datatypes For the end user this is DTFFT_BACKEND_MPI_DATATYPE - backend. But since it does not perform sequence: transpose -> exchange -> unpack, it is internally treated as reshape_handle.
